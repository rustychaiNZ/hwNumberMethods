Number Methods

Key Details

  • Parses a string and returns an interger(whole number)
  • Spaces are allowed
  • Only the first number is returned
  • Will ignore strings of letters and special characters
  • Syntax: parseInt('string'||variable)

Eample

  • First Example: '10.39564 68 years' is the string included
  • I used the variable: var eg1 = '10.39564 68 years';
  • Syntax: document.getElementById('ri1').innerHTML += parseInt(eg1);

Key Details

  • Parses a string and returns a float(number that can include decimal places)
  • Spaces are allowed
  • Only the first number is returned
  • Will ignore strings of letters and special characters
  • Syntax: parseFloat('string'||variable)

Example

  • Second Example: '10.39564 68 years' is the string included
  • I used the variable: var eg2 = '74.38234 45 months';
  • Syntax: document.getElementById('ri1').innerHTML += parseFloat(eg2);
